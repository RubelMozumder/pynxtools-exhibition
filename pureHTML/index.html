overview

explore pynxtools framework

A unified framework to convert, validate, and publish experimental data — from raw vendor files to FAIR NeXus/HDF5.

• 8 reader plugins — SPM, XPS, EM, APM, MPES, ellipsometry, XRD, and Raman
• Validate — checks field completeness, units, and data types against FAIR NeXus standards
• Extend — reuse an existing NeXus datamodel or define a new one with NXDL
• NOMAD-native — semantic search, DOI, and API access without extra configuration
